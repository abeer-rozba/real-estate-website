### Description
#### This is a Node.js web application built with express as part of our Software Engineering Bootcamp program. With Home Sweet Home, users can sign up as either customers searching for their dream homes or agents promoting their properties. The website provides a list of available properties and their relevant information, as well as the contact details of the property's agent. Users can bookmark the properties they like so they can refer back to them later. Agents can create, read, update, and delete properties. This website is like an online catalog of beautiful homes waiting to find their families.

### Technologies Used:

### Getting Started
1. Check out Home Sweet Home through this [live website]().
2. Sign up or sign in to your account, you can only choose one role per account.
3. Search for your dream home or post properties for others to view.
4. If you are a customer, you can view all postings and bookmark your favorite properties and contact agents for further information.
5. If you are an agent, you can view, create, update, or delete properties.
